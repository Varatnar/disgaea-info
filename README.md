# disgaea-info [WIP]

Website to display and easily navigate through the different element of disgaea games.

## Project setup
```
yarn
```

### Build item json asset
```
yarn parse
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Run your unit tests
```
yarn test:unit
```
